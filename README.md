# MyGigPal

A static Progressive Web App (PWA) that scans upcoming gigs from DJ Event Planner and exports them to your calendar.

## Features

- Scan gigs directly from your DJ Event Planner account
- Export gigs to calendar formats (ICS) for import into Google Calendar, Apple Calendar, or any CalDAV client
- Works offline via Service Worker caching
- Installable on iOS and Android as a home screen app

## Usage

Open the app in your browser, log in with your DJ Event Planner credentials, and use the scan and export controls to pull your upcoming gigs into your calendar of choice.

## Tech

Pure static PWA — HTML, CSS, and vanilla JS. No build step required. Deploy to any static host (GitHub Pages, Netlify, Cloudflare Pages, etc.).
