002 — ONE-TIME SETUP

Scan Bridge.

The secure bridge that lets MyGigPal read the planner — without ever seeing your password. A one-time setup.

🔴 Scan Gigs
Drag this up to your bookmarks bar. Then open the planner's Events – Upcoming page and click it.
  1. 1Tap Copy Scan Gigs code above.
  2. 2Tap Safari's Share button (the square with an up-arrow), then Add Bookmark → Save.
  3. 3Tap the Bookmarks icon (open book) → Edit → tap the bookmark you just saved.
  4. 4Clear its web address, paste the copied code, rename it Scan Gigs, then tap Done.
  5. 5To scan anytime: open the planner's Events – Upcoming page, tap the address bar, type Scan Gigs, and tap it.
003 — HOW IT WORKS

Three taps.

  2. 02

  3. 03
    Hit Export. A calendar file downloads. Open it. Done.
